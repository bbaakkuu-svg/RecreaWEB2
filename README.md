# RecreaWEB
ACTIVIDAD 2. Recrea la segunda página web




Tendréis que realizar un pequeño sitio web compuesto por 2 páginas que estarán vinculadas entre sí mediante un menú de navegación.



El sitio web constará de 2 archivos HTML (index.html y news.html), un único archivo CSS (style.css) dentro de una carpeta de nombre css y una carpeta (images) que contendrá los archivos de imagen necesarios.
